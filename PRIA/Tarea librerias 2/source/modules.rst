.. Hospital Management Modules Documentation
=============================================

This section contains documentation for the various modules in the Hospital Management system.

The system consists of the following modules:

1. **Hospital** - The core module for managing hospital workers, including doctors and nurses.
2. **Medico** - The module defining the `Medico` class for managing doctors.
3. **Enfermera** - The module defining the `Enfermera` class for managing nurses.
4. **Trabajador** - The base class for all hospital workers.
